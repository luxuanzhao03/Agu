# A股半自动交易辅助系统实盘检查清单

## 1. 使用说明

这份清单用于实盘或仿真盘前后的标准化检查，建议每天按顺序执行并留痕。  
可打印执行，也可在团队 wiki 中做勾选记录。

---

## 2. 开盘前检查（T 日 08:30-09:25）

## 2.1 系统与数据可用性

- [ ] `start_system_windows.bat` 启动正常，无报错
- [ ] `GET /health` 返回正常
- [ ] 主界面 `/ui/` 显示策略数量正常（非 0）
- [ ] 投研页可正常加载策略列表（`/strategies`）
- [ ] 运维页 `/ops/dashboard` 能刷新成功

## 2.2 连接器与治理健康

- [ ] 连接器新鲜度在阈值内（无明显滞后）
- [ ] `pending_failures` 无异常激增
- [ ] `dead_failures` 无新增或已在处理
- [ ] SLA 未恢复状态数在可控范围
- [ ] 无高等级未确认告警积压

## 2.3 策略与参数准备

- [ ] 今日主策略已确定（例：`trend_following` / `multi_factor` / `event_driven`）
- [ ] 参数模板已核对（与昨日版本一致或已记录变更原因）
- [ ] 日期窗口正确（`start_date/end_date`）
- [ ] `commission_rate/slippage_rate/lot_size` 与账户规则一致
- [ ] 如启用事件增强，`event_lookback_days` 与 `event_decay_half_life_days` 已确认
- [ ] 财报增强保持开启（`enable_fundamental_enrichment=true`）
- [ ] 财报陈旧阈值已确认（`fundamental_max_staleness_days`）

## 2.4 风控边界确认

- [ ] `max_single_position` 不超账户纪律
- [ ] `max_industry_exposure` 不超组合纪律
- [ ] `max_drawdown` 预警线明确
- [ ] 当前组合快照（现金、总资产、回撤）已更新

---

## 3. 盘中执行检查（T 日 09:30-15:00）

## 3.1 信号与结果核对

- [ ] 已运行信号生成并检查动作分布（BUY/SELL/WATCH）
- [ ] 已检查信号表“财报评分”列，低分 BUY 信号已人工复核
- [ ] 已查看风控命中，确认没有忽略 `CRITICAL` 阻断
- [ ] 已看 K 线叠加信号，确认信号点位合理
- [ ] 已查看回测核心指标（收益/回撤/夏普）与最近表现是否一致

## 3.2 组合与调仓核对

- [ ] 研究工作流已运行（多标的）
- [ ] 优化权重图与行业暴露图已检查
- [ ] 调仓建议已生成（`/portfolio/rebalance/plan`）
- [ ] 对调仓单执行了人工合理性审阅（价格、手数、方向）

## 3.3 人工下单纪律

- [ ] 执行前再次检查风险命中摘要
- [ ] 大幅偏离建议成交价的单子有备注原因
- [ ] 未执行建议单有备注原因（流动性、盘口、风险等）

---

## 4. 收盘后检查（T 日 15:00-17:30）

## 4.1 执行回写与复盘

- [ ] 所有已执行交易已完成回写（`/replay/executions/record`）
- [ ] 复盘报表已刷新（`/replay/report`）
- [ ] 已检查跟随率、平均延迟、样本规模
- [ ] 异常执行（未跟随、延迟过大）已记录原因

## 4.2 治理与运维闭环

- [ ] 运维看板再次刷新，确认无新严重 SLA 异常
- [ ] 对当天失败条目完成筛选与必要重放
- [ ] 对 dead-letter 给出处理结论（修复/暂缓/上游问题）
- [ ] 值班回调与告警 ACK 状态已核对

## 4.3 策略复盘与参数变更控制

- [ ] 当日策略表现有简要结论（有效/一般/失效）
- [ ] 若改参数，已记录变更项与预期影响
- [ ] 未经验证的参数变更不进入次日实盘

---

## 5. 每周检查（建议周末）

## 5.1 收益与风险稳定性

- [ ] 回测曲线与实盘回写结果做周度对比
- [ ] 最大回撤、夏普、胜率、阻断数周度统计
- [ ] 执行偏差（跟随率、延迟）周度统计

## 5.2 数据与治理稳定性

- [ ] 连接器 SLA 周报：违约次数、升级次数、恢复时间
- [ ] NLP 漂移周报：命中率趋势、分值分布、贡献变化
- [ ] 失败重放周报：失败类型、修复成功率、死信存量

## 5.3 合规与审计留痕

- [ ] 审计导出链路可用（抽样验证）
- [ ] 必要时导出合规证据包并归档
- [ ] 关键流程（参数变更、异常处置）有记录

---

## 6. 常见红线（命中即暂停加仓）

- [ ] 连接器严重 SLA 且未恢复
- [ ] 关键策略连续多日出现异常回撤
- [ ] 执行跟随率显著下降且原因不明
- [ ] 风控 `CRITICAL` 频繁出现但被人为忽略
- [ ] 财报评分长期偏低但仍持续给出激进买入
- [ ] 事件/NLP 漂移达到高风险且未处理
