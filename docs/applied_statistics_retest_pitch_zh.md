# 应用统计复试展示话术（简版）

## 1. 项目定位（一句话）

这是一个“应用统计研究平台”，金融行情数据只是案例数据源，核心展示的是统计建模与推断流程。

## 2. 可展示的统计流程

1. 数据整理：从原始观测值构建结构化研究样本  
2. 描述统计：均值、方差、分位数、偏度、峰度  
3. 假设检验：组间均值差异（Welch + 置换检验）  
4. 回归建模：OLS 参数估计与显著性检验  
5. 模型诊断：VIF、多重共线性、残差正态性、DW  
6. 区间估计：Bootstrap 置信区间  

## 3. 现场演示建议（5 分钟）

1. 调用 `POST /applied-stats/descriptive` 展示描述统计  
2. 调用 `POST /applied-stats/tests/two-sample-mean` 展示假设检验  
3. 调用 `POST /applied-stats/model/ols` 展示回归显著性  
4. 调用 `POST /applied-stats/cases/market-factor-study` 展示完整案例闭环  

## 4. 结尾说明

本项目保留工程能力（数据获取、审计、接口化），但复试展示重点是“统计思想与统计方法的落地能力”。
